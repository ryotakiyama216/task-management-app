# task-management-app

## プロジェクト概要
Next.js とサーバーレス環境（AWS Lambda + DynamoDB）を使ったタスク管理アプリです。  
- タスクの作成、編集、削除が可能  
- フロントエンド: Next.js (TypeScript + Tailwind CSS)  
- バックエンド: AWS Lambda  
- データベース: DynamoDB

```bash
# リポジトリをクローン
git clone https://github.com/ユーザー名/task-management-app.git
cd task-management-app

# 依存関係をインストール
